# 프로젝트 디렉토리 구조 설계 지침 (Final Instruction)

이 프로젝트는 단순한 웹사이트가 아니라:

```
interactive terminal OS
retro CRT interface
personal archive system
```

성격의 React + Vite 기반 SPA 프로젝트로 설계한다.

구조 설계의 핵심 목표는:

- 유지보수성
- 테마 확장성
- 미디어 관리
- 데이터 아카이브
- 애니메이션 시스템
- 라이브러리 확장
- terminal UX 구조화

이다.

---

